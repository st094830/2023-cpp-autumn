1. поместить папку "memes for Mr.Nesterchuk" в папку репозитория на компьютере
2. в консоли перейти по адресу папки-репозитория
3. git status (необязательно)
4. git add <название>
5. git commit -m <сообщение>
6. git push

Если название папки в задании учитывается, то её надо переименовать. При таком названии командная строка будет ругаться и распознавать for как команду, так что надо, например, поставить нижние почёркивания.
